    # 为什么精英都是时间控

    不要再做时间的奴隶了!

    因为惰性, 让自己贪欲一时的舒适, 结果时间都浪费掉了.

    集中注意力, 提高效率, 可以为自己创造出更多的自由时间, 再把时间用于自我投资, 让自己不断的成长, 能力进一步提高, 让自由时间变得更多, 这样就可以形成一个好的良性循环.

    # 序章: "神之时间管理术" 四原则

    # 第一原则: 以"专注力"为中心分配时间

    为了提高效率, 要灵活运用大脑的黄金时间.

    早上起来, 大脑最清楚, 适合做理论性强 和 高度专注 的工作.

    工作可以分为:

    • 专注性工作
    • 非专注性工作

    在大脑不专注的时候做 "非专注性工作", 费时费力, 非常没效率.

    专注性工作, 放在大脑专注度高的时间里做; 非专注性工作, 放在大脑专注力下降的时间里做.

    很多时候, 人们以为自己可以控制专注力, 但是其实一般人没那个能耐.

    在专注力高的时候做需要高度专注力的工作, 是应用自己专注力的最简单方法. 就像拼拼图, 把相应的工作安排在合适的时间里.

    # 第二原则: 重启专注力, 创造出时间

    一维时间管理:

    把时间看作一维直线, 在地铁上的一小时, 不玩手机, 改成看书. 也就是说, 把 "无意义的时间" 变成 "有意义的时间". 抽象点说 "时间置换".

    这个方法, 不可以突破一天只有24个小时的壁垒.

    二维时间管理:

    把时间管理内看作二维, 横轴是 "时间", 纵轴是 "专注力".

    专注力高的时候, 同样时间可以做更多的工作. 也就是说: 工作量 = 专注力 x 时间

    前面说过提高专注力很难, 但是让专注力下降的少一些还是可以的, 做法是: 在感到疲劳之前及时休息.

    最好的休息是睡觉, 以剥夺睡眠来增加工作量的方法, 只会适得其反.

    有氧运动也是个好办法, 有氧运动会让大脑分泌, BDNF (脑源性神经营养因子)

    # 第三原则: 掌握美国式的工作效率

    专注力很重要, 工作技巧也很重要.

    工作效率提高后, 工作环境也可以更轻松, 人的压力也会减小.

    # 第四原则: 花时间自我投资

    应用时间管理术之后, 不要让自己更忙碌.

    创造出自由时间之后, 不要再投入到工作中去了. 把工作限定在一个固定的时间里, 节省出来的时间用来自我投资, 或者享受人生.
    这样可以提升自己的能力, 让自己拥有 "无限上升成长旋涡"

    # 终极目标: 为了快乐

    管理实践, 让工作高效, 就是为了能够让自己的生活快乐自由.

    为了能够有时间做自己想做的事情, 能够去玩, 就要抓紧时间先把手头的工作完成.

    我要幸福.

    # 第一章: 最大发挥大脑技能, 提高专注力

    # 15.45.90 法则

    人能保持高度集中的平均时间有多久呢? 15分钟, 15分钟可以看做一个 "专注力单位"

    同声传译人员基本上15分钟一轮班; 古典名曲多是15分钟一小章.

    注意力最多可以保持多久? 45分钟, 45分钟可以看做一个 "注意力单位".

    学校上课多是 45分钟 一堂课, 为了让学生保持专注力, 很多有经验的老师把课程分为3个部分, 每15分钟讲个笑话, 让学生休息一下.

    一次工作时间的极限: 90分钟,

    一场足球比赛 90分钟, 同时还要分上下半场. 一部电影也是这么长. 这个时间安排是很合理的.

    无论工作学习, 最好 15分钟 为一个时间单位, 15分钟后小休息一会儿, 45分钟 为一个工作单元, 45分钟后大休息一会儿, 90分钟 可以作为一个工作阶段

    人体有个周期性规律, 叫做 "亚昼夜节律", 以 90分钟 为周期, 清醒度会以 90分钟 为周期发生变化. 简单说, 人清醒的90分钟和疲倦的20分钟会交替到来, 形成循环.

    有了 15, 45, 90 这三个工作时间单位, 我们还要学会按照大脑节奏来进行 "冲浪" 的工作方法.

    在说专注力的时候, 我们要从 "专注力深度" 和 "持续时间" 两个方面着手.

    工作和学习需要的专注力持续时间是不同的, 比如学习一个知识点, 我看了 15分钟 没看懂, 但是我如果休息, 思路就会打断, 这个时候我就可以看 45分钟. 要学会根据工作内容不同, 自己调节所需的时间. 有松有驰.

    # 排除杂念

    专注力的最大敌人 "杂念"

    # 外物造成的杂念

    周遭乱哄哄, 想找东西找不到, 浪费时间在琐事上面. 专注力一旦被打断, 在恢复要 15分钟 至少. 所以让自己的周遭整洁有条理一点.

    总而言之, 不要图一时轻松, 为以后找无穷的麻烦.

    # 思考引起的杂念

    不要让杂念影响专注力, 比如工作时想, 等会下班要吃啥; 客户邮件回了吗, 等等.

    要把待做的事情都写下来, 做出计划, TODO LIST就是个好办法. 有条理的工作计划可以让自己不必去花心里想等会儿做什么.

    人对未完成的课题会记忆深刻, 但完成后就忘的快. 就像电脑完成任务后, 清内存.

    有了计划之后, 对于未完成事情, 自己心里会一直惦念着, 这份紧张感可以鞭策自己快速完成.

    如果杂念难以消除, 一个想法总是来回重复, 挥之不去. 这个就是大脑前额皮质疲惫的信号. 就像我经常疲惫时读书, 一句话总是读不通.

    大脑前额皮质可以对想法进行 "转换". 一种神经递质 "血清素" 可以让前额皮质有活力. 日光浴, 运动, 咀嚼 是补充血清素的好办法

    # 人造成的杂念

    避免被人打扰, 影响专注力.

    # 时间限制工作术

    为什么暑假作业, 总是在最后一天内完成?

    人在紧急情况下, 大脑会分泌 "去甲肾上腺素" 可以提高注意力, 增强学习力.

    用时间来限制自己, 可以激发潜能, 提高效率.

    严守时间限制的工作法, 就是在任何情况下都不能推迟设定好的时间.

    怎么做到呢? 可以在设定好的工作时间之后, 给自己安排别的事情, 自己为了不影响后面工作或事项, 就要逼着自己全身心的投入当前的工作. 比如安排和朋友吃饭, 安排旅游, 或者别的事情.

    # 第二章: 充分利用清晨

    # 起床法

    早上起床后的两个小时是大脑的黄金时间。大脑黄金时间适合做那些需要高度专注的工作。

    重要的事情早上做就不用加班到很晚了。

    但是很多人都说早上起来之后昏昏沉沉的。或者根本就起不了床, 那这该怎么办呢？

    • 方法一: 冲个热水澡, 或者运动一下. 人的白天神经是交感神经。让交感神经处于主导地位。对办法就是让体温升高，心跳呼吸速度加快。
    • 方法二: 开着窗帘睡觉, 让清晨的阳光可以照进来, 阳光照到视网膜可以促进血清素的分泌。血清素让人有活力, 分泌不足容易得抑郁症.
    • 方法三: 就算起不来, 也不要再睡五分钟, 可以做起来睁着眼睛看会儿太阳. 然后运动一小会儿, 出去买个咖啡晒晒太阳是个好办法.
    • 方法四: 多咀嚼吃早点, 前面说咀嚼也可以促进血清素分泌.
    • 方法五: 不要接收多余的信息, 比如看手机, 看电视. 新闻可以晚上看.

    # 早晨最高效的工作方法

    制定好一天的计划, TODO LIST, 有条不紊.

    有了计划更容易进入 "心流"

    # 第三章: 午后重启术

    # 外出吃午餐

    外出吃午餐, 是个恢复专注力的好办法.

    外出吃饭, 可以晒到太阳, 还进行了运动, 还咀嚼了, 可以促进血清素分泌.

    吃饭时就吃饭, 别想太多别的, 或者聊天说话, 这会影响血清素分泌

    外出可以刺激 "场所神经元", 让海马体活跃, 增强记忆力.

    尝试不同的菜品和餐馆, 可以刺激大脑内的 "乙酰胆碱", 这对灵感和创造力有很重要的作用.

    # 小睡技巧

    午睡最好控制在半个小时. 睡前可以喝一点咖啡, 因为咖啡会在30分钟后发挥效力.

    # 下午的重启工作术

    下午2点到4点, 人的工作效率最低, 如何利用十分重要.

    疲劳时可以尝试下面几个技巧恢复专注力:

    • 运动一下会儿, 深蹲, 溜达...
    • 换个工作场景
    • 换个工作内容, 转换一下心情
    • 睡一会, 如果感到很疲惫, 就说明身体已经筋疲力尽了, 要在疲劳感来临之前, 就休息.
    • 别玩手机, 电脑, 这不是休息
    • 最好的办法: 闭目养神, 阻断视觉信息, 让大脑进入α波. 可以试试冥想.

    # 第四章: 把夜晚时间利用到极致的运动 & 睡眠重启术

    # 运动好

    下午疲倦时, 1小时有氧运动可以让大脑恢复清醒, 运动会让大脑神经元网络更活跃, 清晰.

    但不是剧烈运动, 否则疲惫感, 和运动后的低血糖, 会让自己更加没心思工作.

    下午四点到睡觉前三个小时, 这段时间是适合运动的, 但最好是下午四五点左右

    # 不要让压力过夜

    一般人都是以 "一周" 为周期, 周一到周五学习工作, 周末休息娱乐. 但是作者推荐以 "一天" 为周期, 白天工作学习, 晚上休闲娱乐. 这叫做 "不要让压力和疲劳过夜的生活方式"

    这样人才能在第二天一醒来, 精神饱满, 没有多余压力

    多和人交流是最好的休闲方式之一, 聊天时人会分泌 "后叶催产素", 这会让人有爱的感觉. 当然做爱是最好的分泌这种激素的方式.

    不懂得休息的人, 早晚油尽灯枯, 要像重视白天工作一样, 重视晚上的休息

    # 对睡眠极好的习惯

    早晨起床的习惯决定一天, 晚上睡觉的习惯决定一生.

    睡前两个小时千万不要做, 饮酒, 剧烈运动, 洗热水澡, 玩游戏, 看电影等刺激视觉的, 光线太亮

    可以和朋友聊天, 听音乐等

    睡前不要吃东西, 血糖高, 影响生长激素分泌, 会让自己睡很久也不管用. 生长激素可以恢复疲劳

    睡前的 "15分钟" 是记忆的黄金时间, 可以用来背诵

    周末补觉没有用, 还是平时保证睡眠吧

    # 遵守自己的日程规律

    每天按照自己制定好的固定日程计划, 安排时间是让大脑机能发挥最大的生活方式

    身体养成习惯之后, 在固定时间起床, 固定时间睡觉, 工作, 学习, 运动, 严格遵守日程可以让自己能够控制专注力.

    # 第五章: 把工作时间利用到极致的创造时间工作术

    # 为他人工作法

    心里想着他人, 工作时多为别人找想, 不能给别人添麻烦, 可以增加效率, 还能获得尊重. (团队制定好的分工, 时间安排; 约定好的截稿日期; 等等)

    以 "让别人等待的时间" 为基准安排工作的轻重缓急, 浪费别人的时间, 超过规定好的时间, 都是很恶劣的行为.

    遵守时间的人会获得尊重和财富.

    # 提前 30 分钟

    做事情最好提前行动, 这样可以有条不紊, 不会造成延迟

    比如和人约定见面, 提前30分钟不会让自己迟到, 还可以边等边工作, 因为有了时间限制, 效率也会提高. 就算对方迟到, 因为手头有事情, 也不会太在意, 而且还能得到对方信任

    # 趁现在

    什么事情不要拖, 现在能做就做完. 别想着我等会儿再弄, 一来可能你会一拖再拖, 二来之后再做需要一个时间进入状态, 很浪费时间.

    比如回复邮件, 别想明天再说, 因为你明天再回之前还要再读一遍, 这是浪费时间的.

    做决断最好在 30秒内, 想太久也其实不会做出高明很多的结果. 如果 30秒 做不出, 那多是因为手头掌握的信息, 或者情况不允许, 这是要做出 "当 XXX 情况满足时, 我再做决断" 的决断, 即使不知道接下来该怎么做, 也要清楚现在自己欠缺的是什么条件.

    不要给自己设定 "截止日期", 而是设定 "开始日期"

    # 立刻预约

    和客户聊工作的时候, 不要说下次有机会再单独约, 这多半是拉倒了.

    就直接说 "那最近那天有空, 咱们安排见面".

    对方如果真的想见你, 无论多忙也会和你约定时间的, 不想见你会搪塞的. 这样也可以探测对方的想法

    # 并行工作法

    不要一边听音乐一边学习, 或者一边看电视一边工作, 人的大脑是单线程的, 并行干两件需要集中精力的事情, 效率很低

    但是洗澡, 乘车的时候可以看书, 思考问题

    # 第六章: 把自由时间利用到极致的自我投资和自我更新术

    节省出来的自由时间不要用于工作了。想提高工作量一定要从工作效率上着手。单靠增加时间往往事与愿违。

    节约出来的时间可以用于自我投资。个人能力提升的同时, 工作效率也随之提升, 这样就可以节省出更多的时间用于自我投资。

    要选择对自己有益的娱乐方式。爱读书的人容易获得心流体验。天天看电视不容易获得, 看电视容易分散人的注意力。

    需要注意力的娱乐方式叫做主动娱乐。我们应该多选择主动娱乐方式。

    看电视其实也可以变成主动娱乐方式。你可以在节目看完之后向外输出自己得到的知识心得体会。通过这种方式锻炼自己的专注力。

    能够百分之百投入到玩中的人, 也能百分之百的去进行工作。
