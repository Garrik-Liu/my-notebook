# 硬件知识

## 计算机三大核心部件

计算机基础的元件有三个: CPU, 内存, I/O.

* CPU (Central Processing Unit) 中央处理器.  计算机的大脑, 负责解释, 执行程序.
* 内存负责储存程序和数据
* I/O 是输入/输出, 负责将计算机与外部设备连接在一起
