## 连续随机变量

### 什么是连续随机变量

- 在实际生活中可以观察到很多随机变量, 它们的可能取值的个数是不可数的, 因此不属于离散随机变量;
- 这些随机变量的取值可能是一个或多个区间中不可数的无穷多个连续的点中的一个, 这类变量称为『 **连续随机变量** 』
- 我们不可能把有限的概率指派给连续随机变量的所有可能的取值, 并使它们的概率之和为 1. 因为可能取值的数量是不可数的;
- 『 **累积分布函数** 』又称分布函数, 其值等于变量 $Y$ 所允许的最小值, 到一个指定的变量 $Y$ 的可能取值之间所有可能取值对应的概率之和;
  - 表示为 $F(y_0) = P(Y \le y_0)$, $y_0$ 表示一个变量 $Y$ 的可能取值;
  - 对于『 离散随机变量 』, 其计算公式为 $\sum\limits_{y \le y_0}p(y)$
    - $F(y)$ 函数呈现为一个不连续, 成阶梯状的图像;
  - 对于『 连续随机变量 』, 其累积分布函数是一个关于 $y$ 的单调递增连续函数;
    - 若满足 $y_a < y_b$, 则 $F(y_a) \le F(y_b)$

#### 连续随机变量的密度函数

- 在第一章的时候我们将, 对于取值为定量数据的变量, 我们不可能计算出所有的可能取值出现的概率, 但是我们可以给取值范围分出等宽的区间, 然后计算出观测值落入每个区间的概率;
- 我们可以把这个区间的宽度尽可能的缩小, 以至于概率分布的图形呈现出一条光滑的曲线;
- 『 **概率密度函数** 』就是用来计算这条分布曲线的理论模型, 其定义为:
  - 若 $F(y)$ 是连续型随机变量 $Y$ 的累积分布函数, 则 $Y$ 的密度函数 $f(y)$ 是:

$$f(y) = \frac{dF(y)}{dy}$$

![2020-05-31-23-37-08](https://garrik-default-imgs.oss-accelerate.aliyuncs.com/imgs/2020-05-31-23-37-08.png)

- 通过概率密度函数的定义, 还可以得出:

$$F(y) = \int_{-\infin}^y f(t)dt$$

- 从上面的概率密度函数 $f(y)$ 的图形可以看出, $F(y_0)$ 等于 $f(y)$ 函数曲线下由 $-\infin$ 到 $y_0$ 围起来的面积;
